Detailed Description

The Renesas FAT FS Abstraction Layer.

Summary

This middleware module contains an API Abstraction for the FAT File System Library allowing for portability between different FAT File System Implementations.

Known Implementations:
This driver is used in the RZA1LU Software Package.
See also
RENESAS_APPLICATION_SOFTWARE_PACKAGE
Operating System Abstraction module Renesas OS Abstraction interface
