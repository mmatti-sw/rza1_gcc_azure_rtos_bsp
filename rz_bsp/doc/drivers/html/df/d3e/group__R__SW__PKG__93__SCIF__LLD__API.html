Detailed Description

The API for the low level SCIF driver.

Summary

This interface defines the functionality that the low level driver must implement for compatibility with the high level driver.

This is a device and board-specific implementation of an SCI driver, configured for the RZA1LU Stream-IT board

Known Limitations
None

Known Implementations:
This driver is used in the RZA1LU Software Package.
