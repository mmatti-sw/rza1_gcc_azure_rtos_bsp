Detailed Description

This interface provides access to SCIF Peripheral.

Summary

The driver implements High Level access to the SCIF Peripheral.

Before accessing the driver, call the open function to configure the selected hardware interface.

When finished with the driver close it using the close function.

This driver does not allocate any heap memory.

This driver does not use locking primitives.

Driver Instance : g_scif_driver

Driver Implementation:

"SCIF Device Driver" - Driver Name
scif_open - Opens the RIIC Driver
scif_close - Closes the RIIC Driver
scif_read - Read from SCI
scif_write - Write to SCI
scif_control - see _CTLCODE for commands
Supported:
  • CTL_SCI_SET_CONFIGURATION See _SCICFG for parameter details.
  • CTL_GET_RX_BUFFER_COUNT
scif_get_version - Get driver version

Generic example of driver usage st_def_stdio_device_t

This SCIF High Level Driver is developed for the RZA1LU Stream-IT board.

Known Implementations:
This driver is used in the RZA1LU Software Package.
