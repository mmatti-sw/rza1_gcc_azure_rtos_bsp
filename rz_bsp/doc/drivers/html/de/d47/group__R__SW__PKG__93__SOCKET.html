Detailed Description

A driver to integrate lwIP into the POSIX style API.

Summary

A driver to integrate lwIP into the POSIX style API framework for functions close, read and write which conflict the C run time library functions in lowsrc.c.

Known Implementations:
This driver is used in the RZA1LU Software Package.
