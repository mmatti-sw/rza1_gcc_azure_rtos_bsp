Detailed Description

Sound module responsible for setting up the Audio Codec.

Summary

This middleware module is responsible for configuring the Audio Codec, using the I2C driver to communicate with the MAX9856 Registers.

Known Implementations:
This driver is used in the RZA1LU Software Package.
