Detailed Description

This interface provides low level device specific access to a Watchdog timer WDT device on the RZA1LU Stream-IT board.

Summary

This interface defines the functionality that the low level driver must implement for compatibility with the high level driver.

Known Limitations
NONE_YET

Known Implementations:
This driver is used in the RZA1LU Software Package. RENESAS_APPLICATION_SOFTWARE_PACKAGE
