enum e_colour_code_pmod_lcd_t

DISPLAY COLOUR DEFINITIONS (16 bits) R5G6B5 format.

Only primary & secondary colours are defined here. Other colours can be created using RGB values. 0x00RRGGBB
Example 0x00FFFFFF is White (High byte is never used, RR = 0xFF, GG = 0xFF, BB = 0xFF)

◆ e_ctrl_code_pmod_lcd_t

enum e_ctrl_code_pmod_lcd_t

The PMOD control functions
When using this driver please remember the following process

call the control function CTL_PMOD_LCD_R_LOCK to grab control of the display
call the desired image DISPLAY or DRAW control functions to update the display's buffer
call the control function CTL_PMOD_LCD_UPDATE_DISPLAY to update the screen (if required)
call the control function CTL_PMOD_LCD_R_UNLOCK to release control of the display allowing others to update

Enumerator
CTL_PMOD_LCD_R_DISPLAY_ON 

Enable Display (display can be written to whiles disabled)
pCtlStruct is not used as there is no associated data with this control.

CTL_PMOD_LCD_R_DISPLAY_OFF 

Disable Display (display can be written to whiles disabled)
pCtlStruct is not used as there is no associated data with this control.

CTL_PMOD_LCD_R_LOCK 

Grabs screen lock, task holding lock should update the display
pCtlStruct is not used as there is no associated data with this control.

CTL_PMOD_LCD_R_UNLOCK 

Release screen lock
pCtlStruct is not used as there is no associated data with this control.

CTL_PMOD_LCD_UPDATE_DISPLAY 

This function puts a 24-bit image on the 16-bit display,
colour reduction is embedded in display routine.
pCtlStruct is not used as there is no associated data with this control.

CTRL_PMOD_LCD_DRAW_BMP_IMAGE 

Send bitmap Image (including BMP header) to with header to back buffer, call refresh to show
pCtlStruct uses the st_image_def_pmod_lcd_t data structure

CTRL_PMOD_LCD_DRAW_RAW_IMAGE 

Send bitmap Image to with header to back buffer,
call refresh to show
pCtlStruct uses the st_image_def_pmod_lcd_t data structure

CTRL_PMOD_LCD_DISPLAY_STRING_AT_LINE 

This function controls LCD writes to the start of the line of the LCD.
specified by position of the LCD. If position is greater than the
number of lines available then the LCD write will be to the last line.
pCtlStruct uses the st_line_display_pmod_lcd_t data structure

CTRL_PMOD_LCD_DISPLAY_STR 

Display the string at current cursor position
pCtlStruct uses the uint8_t* data structure

CTRL_PMOD_LCD_DISPLAY_CENTER 

Display the sting at the center of the specified line
pCtlStruct uses the st_line_display_pmod_lcd_t data structure

CTRL_PMOD_LCD_DISPLAY_CLEAR 

Clear the entire display
pCtlStruct uses the uint32_t data structure

CTRL_PMOD_LCD_DISPLAY_CLEAR_LINE 

Clear a specified line
pCtlStruct uses the uint32_t data structure

CTRL_PMOD_LCD_DISPLAY_SET_CURSOR 

Set the current cursor position
pCtlStruct uses the st_cursor_pos_pmod_lcd_t data structure

CTRL_PMOD_LCD_DISPLAY_SET_FONT_COLOUR
