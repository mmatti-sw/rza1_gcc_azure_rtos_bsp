Detailed Description

Timer module which allows user to start and stop a timer, returning the time elapsed in seconds between calls.

Summary

This middleware module provides basic timer functionality allowing the user to trigger a timer to start and stop, returning the time elapsed between calls, in seconds.

Known Implementations:
This driver is used in the RZA1LU Software Package.
