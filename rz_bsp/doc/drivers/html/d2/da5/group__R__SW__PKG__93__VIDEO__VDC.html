Detailed Description

VDC Unit Module.

Summary

This middleware module provides functionality for a user to utilise the Video Display Controller (VDC) for graphical/video output onto a display. Configuration settings range from enabling interrupts to alpha blending.

Known Implementations:
This driver is used in the RZA1LU Software Package.
See also
