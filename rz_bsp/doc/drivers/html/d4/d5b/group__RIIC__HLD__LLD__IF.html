Detailed Description

This is the interface of callback functions for use low level ISRs for the RIIC driver.

Summary

This file provides function definitions for the high-level functions to handle conditions such as receive buffer full, transmit buffer empty, transmit end. They should be connected to the relevant interrupt handlers in the low level driver.
