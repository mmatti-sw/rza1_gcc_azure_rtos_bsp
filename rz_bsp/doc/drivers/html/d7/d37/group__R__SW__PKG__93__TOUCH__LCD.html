Detailed Description

Touch Panel LCD Controller.

Summary

This is a LCD Controller middleware module, which allows the user to write to and read from the registers of the LCD Display.

This module is specific to the LCD panel, but not the device or board.

Known Implementations:
This middleware is used in the RZA1LU Software Package.
