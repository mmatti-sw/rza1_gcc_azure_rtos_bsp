Detailed Description

This interface provides access to the Serial Sound Interface SSIF peripheral.

Summary

The driver implements High Level access of the SSIF Peripheral

Before accessing the driver, call the open function to configure the selected hardware interface.

When finished with the driver close it using the close function.

This driver does not allocate any heap memory.

This driver does use locking primitives for access control.

Driver Instance : g_ssif_driver

Driver Implementation:

"SSIF Device Driver" - Driver Name
ssif_open - Opens the SSIF Driver
ssif_close - Closes the SSIF Driver
ssif_read - Reads from the SSIF Peripheral
ssif_write - Writes to the SSIF Peripheral
ssif_control - see e_control_codes_ssif_t for commands
Supported:
  • R_SSIF_CONTROL_CONFIGURE Configure SSIF, uses parameter st_r_ssif_drv_control_t
  • R_SSIF_CONTROL_STATUS Report SSIF status, uses parameter st_r_ssif_drv_control_t
  • R_SSIF_AIO_READ_CONTROL Configure SSIF to read, uses parameter aiocb
  • R_SSIF_AIO_WRITE_CONTROL Configure SSIF to write, uses parameter aiocb
ssif_get_version - Get driver version
