Detailed Description

Summary

This is a simple ADC Low Level Driver developed for the RZA1LU Stream-IT board.

This is a device and board specific implementation of an ADC driver, configured for the RZA1LU Stream-IT board.

Known Implementations:
This driver is used in the RZA1LU Software Package.
