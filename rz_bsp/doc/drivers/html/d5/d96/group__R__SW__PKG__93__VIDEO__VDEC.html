Detailed Description

VDEC Unit Module.

Summary

This middleware module provides functionality for a user to utilise the Video Decoder EnCoder (VDEC) unit, for detecting the colour format of a video input.

Known Implementations:
This driver is used in the RZA1LU Software Package.
