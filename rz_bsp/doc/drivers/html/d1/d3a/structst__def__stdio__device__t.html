Detailed Description

This reference provides a demonstration of using High Level drivers via POSIX/STDIO and also by direct call, using the OSTM driver as a specific example.

ANSI STDIO Call Example open
   // DEVICE_INDENTIFIER          "\\\\.\\"
   static int_t gs_handle = DRV_ERROR;
   // Open the ostm driver
   gs_handle  = open(DEVICE_INDENTIFIER "ostm", 0);
   // Only continue if the drive has successfully been opened
   configASSERT((DRV_ERROR != gs_handle));
Direct Call Example open
   static int_t gs_handle = DRV_ERROR;
   // Open the ostm driver
   gs_handle = g_stdio_driver.p_device.open(NULL);
   // Only continue if the drive has successfully been opened
   configASSERT((DRV_ERROR != gs_handle));
ANSI STDIO Call Example close
   // gs_handle MUST have been opened
   // Close the ostm driver
   close(gs_handle);
Direct Call Example close
   // gs_handle MUST have been opened
   // Close the ostm driver
   gs_handle = g_stdio_driver.p_device.close(gs_handle);
